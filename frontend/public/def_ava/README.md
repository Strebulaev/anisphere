# Дефолтные аватарки пользователей

Поместите ваши JPG/PNG изображения в эту папку. После сборки они автоматически попадут в `dist/def_ava/`.

## Что нужно сделать

1. **Загрузите свои аватарки** в папку `frontend/public/def_ava/`
   - Форматы: JPG, PNG, WEBP
   - Названия: `def_1.jpg`, `def_2.jpg`, `def_3.png` и т.д.

2. **Обновите список файлов** в `frontend/src/utils/defaultAvatars.ts`:
   ```typescript
   const AVATAR_FILES = [
     'def_1.jpg',
     'def_2.jpg',
     'def_3.png',
     // Добавьте имена ВСЕХ ваших файлов
   ]
   ```

3. **Соберите проект**: `npm run build`
   - Файлы из `public/def_ava/` попадут в `dist/def_ava/`

## Готово!

При регистрации новым пользователям будет автоматически назначаться случайная аватарка из списка.
