# Аватарки озвучек

В этой папке хранятся аватарки для озвучек аниме.

## Правила именования

Имя файла должно соответствовать названию озвучки в нижнем регистре, с заменой пробелов на дефисы и удалением специальных символов.

Примеры:
- Озвучка "Anilibria" → `anilibria.png`
- Озвучка "Studio Band" → `studio-band.png`
- Озвучка "JAM" → `jam.png`

## Поддерживаемые форматы

- PNG (рекомендуется)
- JPG/JPEG
- WEBP

## Размер

Рекомендуемый размер аватарки: 200x200px
Минимальный размер: 100x100px

## Пример использования

Если у озвучки есть аватарка в этой папке, она будет автоматически отображаться.
Если аватарка не найдена, будет показано изображение по умолчанию или инициалы.
