# Аватарки озвучек - Инструкция

## Что сделано

Создана система автоматического отображения аватарок для озвучек. Если у озвучки есть логотип в API, он отображается первым. Если нет - система ищет аватарку в папке `frontend/src/assets/translation-avatars/`.

## Созданные аватарки

Уже созданы заглушки для популярных озвучек:
- `anilibria.svg` - Anilibria
- `studio-band.svg` - Studio Band
- `jam.svg` - JAM
- `fanvadub.svg` - FanVADub
- `coldfilm.svg` - ColdFilm
- `shiza-project.svg` - Shiza Project
- `kansai.svg` - Kansai
- `animedub.svg` - AniDub
- `zendub.svg` - ZenDub
- `default.svg` - Заглушка по умолчанию

## Как добавить аватарку для новой озвучки

### 1. Узнайте название озвучки
Откройте страницу аниме и посмотрите список озвучек.

### 2. Нормализуйте название
Система автоматически преобразует название:
- Переводит в нижний регистр
- Заменяет пробелы на дефисы
- Удаляет специальные символы
- Транслитерирует кириллицу

Примеры:
- "Anilibria" → `anilibria.svg`
- "Studio Band" → `studio-band.svg`
- "Дубляж" → `dublyazh.svg`
- "JAM" → `jam.svg`

### 3. Создайте файл
Поместите файл в папку `frontend/src/assets/translation-avatars/` с нормализованным названием.

### 4. Поддерживаемые форматы
- SVG (рекомендуется для заглушек)
- PNG (рекомендуется для реальных аватарок)
- JPG/JPEG
- WEBP

### 5. Рекомендуемый размер
- 200x200px для PNG/JPG
- Любой размер для SVG (масштабируется автоматически)

## Примеры

Если у вас есть озвучка "Мой Дуб" (My Dub), создайте файл:
```
frontend/src/assets/translation-avatars/moy-dub.svg
```

Если у вас есть озвучка "Amazing Team", создайте файл:
```
frontend/src/assets/translation-avatars/amazing-team.svg
```

## Как узнать, какой файл создать

1. Откройте консоль разработчика (F12)
2. Найдите озвучку в списке
3. Посмотрите название озвучки
4. Нормализуйте его по правилам выше
5. Создайте файл с этим названием

## Замена существующих аватарок

Если вы хотите заменить существующую аватарку:
1. Найдите файл с нормализованным названием озвучки
2. Замените его новым файлом
3. Обновите страницу

## Примечания

- Если аватарка не найдена, отображаются инициалы озвучки
- Система сначала проверяет логотип из API, затем ищет в assets
- Аватарки кэшируются браузером для быстрой загрузки
