| Tier | Badge |
|---|---|

## 3. Complaints & Moderation

### Complaint Hierarchy
1. Users → report content (broken link, rule violation)
2. Verified dubbing studios → mass-report illegal links
3. Automated system → domain blacklist checks (RKN registry)

### Report Reasons
`Спам` · `Нарушение авторских прав` · `Оскорбления/травля` · `Неприемлемый контент (18+)` · `Другое`

### Report Statuses
`На рассмотрении` → `Принята (контент удалён)` | `Отклонена` | `Требуется дополнительная информация`

### Moderator Actions
- Issue warning to author
- Hide content
- Delete content
- Temporary ban
- Permanent ban

### Toxicity Escalation
Warning → Temp ban → Permanent ban → Appeals committee review
